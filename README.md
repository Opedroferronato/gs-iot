# Sistema Inteligente de Detecção de Fadiga para Logística

## Descrição do Projeto

Este projeto foi desenvolvido com o objetivo de aplicar conceitos de Visão Computacional em um contexto logístico real.

A solução consiste em um sistema inteligente de monitoramento de fadiga de motoristas utilizando webcam em tempo real. A aplicação utiliza inteligência visual para identificar sinais de sonolência e distração, ajudando na prevenção de acidentes e atrasos logísticos.

O sistema realiza:

* Captura de vídeo em tempo real;
* Detecção facial;
* Monitoramento dos olhos;
* Identificação de sonolência;
* Detecção de bocejos;
* Alertas visuais de risco.

---

## Problema

Na logística moderna, acidentes causados por fadiga de motoristas geram:

* atrasos;
* prejuízos financeiros;
* aumento de custos operacionais;
* riscos humanos;
* perda de eficiência logística.

---

## Solução

A proposta do projeto é utilizar Visão Computacional para detectar sinais de fadiga em tempo real, permitindo identificar situações de risco antes que acidentes ocorram.

---

## Tecnologias Utilizadas

* Python
* OpenCV

* NumPy

## Funcionalidades

* Captura da webcam em tempo real;
* Rastreamento facial;

* Detecção de bocejo;
* Sistema de alerta de fadiga;
* Interface visual simples em tempo real.
